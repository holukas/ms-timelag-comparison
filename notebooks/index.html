The analysis pipeline, in order. Each notebook reads the previous stage’s output and writes the next (see Data for the folder layout). Click through to the rendered notebook.

  1. 01 · Read EddyPro FLUXNET CSVs → Parquet. Reads each EddyPro FLUXNET CSV with diive and saves it as Parquet (00- → 01-).

  2. 02 · Subset flux columns. Keeps a defined column list (fluxes and time-lag diagnostics) (01- → 02-).

  3. 03 · Plot fluxes and time lag used. Plots flux over time lag used per analyzer and gas, by variant, into figures/03_*.png.

  4. 04 · Subset flux product FP2025.3 (year 2024). Extracts the 2024 reference subset from the external FP2025.3 product (→ 04-).

  5. 05 · Merge variants with the flux product. Merges all variant subsets with the 2024 reference, suffixed by source (→ 05-).
